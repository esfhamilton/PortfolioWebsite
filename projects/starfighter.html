Starfighter

Description

Starfighter is a modern version of the arcade game "Asteroids" designed with more of an endless/ arena based setting and various twists.

This was created for my Computer Science project during college and was wrote in Unity using C#. Further developments were intended such as a shop system for players to upgrade their spaceship. However, this was the submitted product.

To play the game: Click the button below, extract the files from the zip folder and run Starfighter.exe. Ensure Graphics Quality is set to "Fantastic" to make sure the game runs correctly.
